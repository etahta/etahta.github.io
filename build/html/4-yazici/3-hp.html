Hp Yazıcı¶

Hp 1005-1018-1020 Yazıcıları kurmak için;

  • Aşağıdaki komutları çalıştırmanız gerekmetedir.

sudo apt-get update

sudo apt-get install printer-driver-all

sudo apt-get install printer-driver-foo2zjs printer-driver-foo2zjs-common

sudo apt-get install hplip

sudo hp-plugin

  • Komutlar hatasız çalıştırıldıktan sonra cups üzerinden kurulum gerçekleştirebilirsiniz.

  • Buradan yapamayanlar etahta içerisinde bulunan Betik Yükleyici kullanarakda kurulum yapabilirler.

  • Sorun yaşayanlar aşağıdaki sayfayı ziyaret edebilirler.

https://forum.pardus.org.tr/t/hp-laserjet-m1132-mfp-yazici-sorunu/16567/5
